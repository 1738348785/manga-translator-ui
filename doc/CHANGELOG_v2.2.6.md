# v2.2.6 更新日志

发布日期：2026-04-02

## 🔧 优化

- 优化桌面编辑器局部修复逻辑，连续涂抹时只应用最新一次修复结果。

## 🐛 修复

- 修复编辑器实时修复在快速操作时可能出现的结果回退问题。
- 减少编辑器蒙版修复过程中的额外卡顿。
- 修复字体管理器预览区在切换同家族不同样式字体时显示不准确的问题，避免预览错误合并为同一字体效果。
- 修复渲染器字体选择逻辑，按当前选中的具体字体文件取字形，正确区分不同字体家族及同家族下的不同样式。

## 📝 说明

- 本版本主要包含桌面编辑器体验修正，以及字体预览与渲染字体选择的兼容性修复，没有新增独立功能入口。
